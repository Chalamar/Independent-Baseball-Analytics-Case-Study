Chalamar Elkridge Jr. – Run Prevention Analysis & Game Plan

This folder contains a pitcher-focused run prevention analysis and a hitter-specific game plan developed as part of a technical assessment.

Files:
- Chalamar_Run_Prevention_Analysis.pdf  
  Final presentation summarizing pitch mix changes, effectiveness metrics, and a game plan for Pitcher B vs Player A.

- pitcher_analysis.R  
  R script used to analyze Pitcher B’s pitch mix, whiff, CSW, command, and velocity trends.

- batter_analysis.R  
  SQL-style queries (executed via SQLite in R) and R code used to analyze Player A’s swing behavior, damage profile, and pitch-type tendencies.

All visuals were generated in R using ggplot2.
